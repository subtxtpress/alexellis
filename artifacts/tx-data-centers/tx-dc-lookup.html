How to use TX SOS

Go to sos.state.tx.us → SOSDirect → Search by Entity Name → enter the LLC name → click the entity → look at Registered Agent address and Principal Office address. The principal office address is often the actual facility location for single-purpose LLCs. If the address is a registered agent service (CT Corporation, Capitol Services, etc.), try the filing documents tab — the original Certificate of Formation sometimes lists the physical address.
