Key constraint: #C04E01 (burnt orange) on #060B14 yields 4.1:1 — just below the 4.5:1 AA threshold for normal body text. It passes for large text (≥18px bold / ≥24px regular) and UI components (3:1 minimum). Rule for the full redesign: orange is structural only — G03 gradient border accents, the header blockquote left-rule, and Critical Finding card borders. Labels/eyebrow use #EFD7AB (14.6:1 AAA). Never orange in body copy. Additionally, orange on teal (#12403C) is 2.4:1 — never use for text in any context.
